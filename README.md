# dotfiles
## Personal Arch Linux configuration files


Configuration files for i3, i3-blocks, nvim, bash, alacritty, fontconfig and ranger. 
i3-blocks requires scripts as well, which are installed to .local/bin/scripts in accordance with the XDG base directory spec.
More details on the spec can be found [here](https://specifications.freedesktop.org/basedir-spec/basedir-spec-latest.html).
